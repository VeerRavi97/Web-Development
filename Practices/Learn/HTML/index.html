Div and Span

div and span allow group together. useful in css
div: block level element
span: inline element

Block and Inline Element

Block Element
A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can). eg. div, h1-h6, footer,header, section,p,ol,ul, li,nav, form,pre, section,table
Inline Elements

An inline element does not start on a new line and only takes up as much width as necessary.
eg. span, a,btn, img, input, span, textarea
